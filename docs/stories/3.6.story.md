## Story

**As a** developer (Fernando),
**I want** all agents callable as n8n workflow nodes,
**so that** the entire pipeline can be orchestrated visually in n8n.

## Dev Notes

### Agent Server na AZ_Net
[Source: architecture/copyzen-architecture.md#5.1]
O agent server (`http://copyzen-agents:3001`) é acessível apenas dentro da rede Docker AZ_Net pelo n8n. Nunca expor publicamente. Traefik NÃO roteará para este serviço.

### Dockerfile Pattern
```dockerfile
FROM node:20-alpine AS builder
WORKDIR /app
COPY package*.json ./
COPY packages/agents/package*.json ./packages/agents/
RUN npm ci --workspace=packages/agents
COPY packages/agents ./packages/agents
COPY packages/core ./packages/core
RUN npm run build --workspace=packages/agents

FROM node:20-alpine
WORKDIR /app
COPY --from=builder /app/packages/agents/dist ./dist
COPY --from=builder /app/node_modules ./node_modules
EXPOSE 3001
CMD ["node", "dist/server.js"]
```

### Env Vars do Agent Server
```
CLAUDE_API_KEY=
CLAUDE_MODEL=claude-sonnet-4-6
SUPABASE_URL=
SUPABASE_SERVICE_ROLE_KEY=
AGENT_API_KEY=  # chave compartilhada com n8n
GOOGLE_SERVICE_ACCOUNT_JSON=  # base64
PORT=3001
```
