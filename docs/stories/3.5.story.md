## Dev Notes

### Image Prompt Structure
[Source: PRD#3.5-AC3, FR-19]
O prompt para Nano Banana/Kling AI deve incluir:
- Estilo visual: "{heading_font} typography, {primary_color} as dominant color, {tone_of_voice} mood"
- Composição: baseado no format (post quadrado, carrossel, hero wide, etc.)
- Referências: "inspired by [brand reference images if available]"
- Evitar: "generic stock photo, corporate, bland"

### CSS Custom Properties Pattern
```css
--brand-primary: {primary_color};
--brand-secondary: {secondary_color};
--brand-accent: {accent_color};
--brand-bg: {background_color};
--brand-text: {text_color};
--brand-heading-font: '{heading_font}', sans-serif;
--brand-body-font: '{body_font}', sans-serif;
```

### Google Drive Auth
[Source: PRD#3.5-AC5, NFR-06]
Usar service account JSON (não OAuth pessoal). Arquivo JSON via env var `GOOGLE_SERVICE_ACCOUNT_JSON` (base64 encoded). Nunca commitar o arquivo de credenciais.
