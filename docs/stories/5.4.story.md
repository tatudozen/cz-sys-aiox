## Story

**As a** lead (captured),
**I want** to access exclusive content by sharing with friends,
**so that** I get VIP access while helping the professional grow their audience.

### VIP Access Logic
[Source: PRD#5.4-AC4, FR-14]
```
1. Lead B chega via ?ref=CODE
2. System: valida CODE → encontra Lead A
3. Lead B faz cadastro (retencao) com referral_code de A no payload
4. n8n ou API route: lead_a.referral_count++
5. Se referral_count >= vip_threshold (3): lead_a.vip_access = true
6. GHL: addTag(leadA_contact_id, 'vip-access')
7. Lead A recebe WhatsApp: "Parabéns! Você ganhou acesso VIP 🎉"
```
