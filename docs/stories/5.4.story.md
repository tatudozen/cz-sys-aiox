# Story 5.4: Página de Transformação (T) & Referral Tracking

## Story

**As a** lead (captured),
**I want** to access exclusive content by sharing with friends,
**so that** I get VIP access while helping the professional grow their audience.

---

## Status

Approved

---

## Context

**Epic 5 — Sistema 2: FunWheel (A-R-T)**

Dependência de 5.3 (leads capturados com referral_code). Implementa a mecânica de indicação viral — leads ganham acesso VIP após indicar X pessoas.

---

## Acceptance Criteria

- [ ] **AC-1:** Template `transformacao.astro` com:
  - Estado condicional: verificar se lead veio via indicação (`?ref=`) ou já tem acesso VIP
  - Se não tem acesso: mostrar mecânica de compartilhamento + contador de indicações
  - Se tem acesso VIP: redirect para comunidade GoHighLevel

- [ ] **AC-2:** `referral_code` da Story 5.3 usado para gerar link de compartilhamento: `{funwheel_base_url}/r?ref={referral_code}`

- [ ] **AC-3:** Tracking: quando lead B se cadastra via `?ref=CODE`, incrementar `referral_count` do lead A + criar registro em tabela `referrals`:
  - `id`, `referrer_lead_id`, `referred_lead_id`, `created_at`

- [ ] **AC-4:** Quando lead atinge meta de indicações (default: 3, configurável no brand config):
  - `vip_access = true` no Supabase
  - GoHighLevel: adicionar tag `vip-access` ao contato
  - Disparar automation de boas-vindas VIP
  - Redirect para comunidade/área de membros GoHighLevel

- [ ] **AC-5:** Share buttons: WhatsApp (prioritário), link copiável, Instagram Stories (opcional)

- [ ] **AC-6:** Teste: lead A → compartilha → lead B cadastra via ref → A ganha crédito → após 3 indicações, VIP liberado

### VIP Access Logic
[Source: PRD#5.4-AC4, FR-14]
```
1. Lead B chega via ?ref=CODE
2. System: valida CODE → encontra Lead A
3. Lead B faz cadastro (retencao) com referral_code de A no payload
4. n8n ou API route: lead_a.referral_count++
5. Se referral_count >= vip_threshold (3): lead_a.vip_access = true
6. GHL: addTag(leadA_contact_id, 'vip-access')
7. Lead A recebe WhatsApp: "Parabéns! Você ganhou acesso VIP 🎉"
```
