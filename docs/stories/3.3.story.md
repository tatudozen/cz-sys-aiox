## Dev Notes

### Agent Base Class
```typescript
abstract class Agent {
  protected llm: LLMProvider;
  protected supabase: SupabaseClient;

  constructor(config: AgentConfig) {
    this.llm = createLLMProvider(config.llm);
    this.supabase = createSupabaseClient(config.supabase);
  }

  protected async callLLM(systemPrompt: string, userPrompt: string): Promise<LLMResponse> {
    return this.llm.complete(userPrompt, { systemPrompt });
  }
}
```

### Agent Server Internal Only
[Source: architecture/copyzen-architecture.md#5.1]
O agent server roda na rede Docker AZ_Net, acessível apenas pelo n8n internamente. URL: `http://copyzen-agents:3001`. Sem exposição pública.
