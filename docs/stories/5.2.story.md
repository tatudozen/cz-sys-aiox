# Story 5.2: Página de Apresentação (A)

## Story

**As a** lead (visitor),
**I want** to be engaged by a transformation narrative page,
**so that** I understand how this professional can help me and I want to learn more.

---

## Status

Approved

---

## Context

**Epic 5 — Sistema 2: FunWheel (A-R-T)**

Dependência de 5.1 (templates base), 3.4 (CopywriterAgent). Primeira página do FunWheel — engaja o lead com storytelling e direciona para Retenção.

---

## Acceptance Criteria

- [ ] **AC-1:** Template `apresentacao.astro` com seções:
  - Hero (headline + subheadline)
  - Problema (dor do público-alvo)
  - Jornada de transformação (storytelling em 3-4 etapas)
  - Solução (o que o profissional oferece)
  - CTA proeminente → URL da página de Retenção

- [ ] **AC-2:** Copy gerado pelo CopywriterAgent com `generateLandingPageCopy(brief, brandConfig, 'apresentacao')`

- [ ] **AC-3:** Scroll-driven storytelling: seções se revelam conforme scroll (CSS animations, sem JS pesado — Astro islands se necessário)

- [ ] **AC-4:** CTA repetido: topo (após hero) e fundo (após seção solução)

- [ ] **AC-5:** Meta tags SEO: `<title>`, `<meta description>`, `og:image`, `og:title`

- [ ] **AC-6:** Funcional e responsiva: mobile, tablet, desktop

- [ ] **AC-7:** Lighthouse > 90 com copy real
