## Dev Notes

### Deploy Targets
[Source: architecture/copyzen-architecture.md#2.2]
- **KVM4 (CopyZen):** build local → rsync → `/srv/sites/fw.copyzen.com.br/` → nginx serve → Traefik SSL
- **Vercel (clientes):** build → `vercel deploy --prod` → CDN global → SSL auto

### Env Vars por Target
```
# KVM4
FUNWHEEL_BASE_URL=https://fw.copyzen.com.br
API_BASE_URL=http://copyzen-agents:3001  # interno

# Vercel
FUNWHEEL_BASE_URL=https://{client_slug}.vercel.app
API_BASE_URL=https://api.copyzen.com.br  # ou internal se VPN
```
