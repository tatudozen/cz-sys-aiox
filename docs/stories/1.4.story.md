# Story 1.4: Docker Swarm Stack para CopyZen Site

## Story

**As a** operator (Fernando),
**I want** a nginx stack compose file ready for Portainer deployment serving `fw.copyzen.com.br`,
**so that** the CopyZen FunWheel is served via nginx with Traefik SSL on the existing Swarm.

## Context

**Epic 1 — Foundation & Infrastructure Setup**

Dependência de 1.1 (monorepo), 1.3 (CI/CD para rsync). Esta story é puramente de infraestrutura — cria os arquivos Docker Compose para serem deployed manualmente via Portainer UI na KVM4.

**Infraestrutura existente na KVM4:**
- Docker Swarm rodando
- Traefik como reverse proxy com certresolver `letsencryptresolver`
- Rede `AZ_Net` (overlay Docker Swarm)
- nginx:alpine disponível
- IP da KVM4: `31.97.26.21`
- Portainer para gestão de stacks

**Nota:** Esta story NÃO faz deploy automaticamente — cria os artefatos para deploy manual via Portainer.

## Dev Notes

### Infraestrutura KVM4
[Source: architecture/copyzen-architecture.md#2.2]
- KVM4: 4 CPU, 16 GB RAM, IP 31.97.26.21, Debian 11
- Recursos alocados ao CopyZen nginx: 0.1 CPU, 128 MB RAM
- Rede Docker Swarm: AZ_Net (overlay, external ao stack)
- Traefik já configurado com certresolver `letsencryptresolver`

### CRÍTICO: Portainer UI Only
[Source: PRD#NFR-15]
Deploy de stacks Docker na KVM4 é EXCLUSIVAMENTE via Portainer UI. O arquivo `.yml` é o artefato — Fernando faz o deploy manualmente copiando/colando no Portainer.

### Compose File Pattern
```yaml
version: '3.8'
services:
  copyzen-fw:
    image: nginx:alpine
    volumes:
      - /srv/sites/fw.copyzen.com.br:/usr/share/nginx/html:ro
    networks:
      - AZ_Net
    deploy:
      resources:
        limits:
          cpus: '0.1'
          memory: 128M
      restart_policy:
        condition: on-failure
    labels:
      - "traefik.enable=true"
      - "traefik.http.routers.copyzen-fw.rule=Host(`fw.copyzen.com.br`)"
      - "traefik.http.routers.copyzen-fw.entrypoints=websecure"
      - "traefik.http.routers.copyzen-fw.tls.certresolver=letsencryptresolver"
networks:
  AZ_Net:
    external: true
```
