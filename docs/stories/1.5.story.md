**so that** I can confirm the entire pipeline (code → build → transfer → serve) works before building real features.

## Dev Notes

### Página Placeholder Requirements
[Source: PRD#3.1, architecture/copyzen-architecture.md#3]
- Stack: Astro 5.x + Tailwind CSS 4.x
- Zero JS runtime no cliente (Astro static)
- Mobile-first — breakpoints: < 768px, 768-1024px, > 1024px
- Lighthouse > 90 em todos os 4 critérios

### Pipeline End-to-End
[Source: architecture/copyzen-architecture.md#2.5]
```
git push main
  → GitHub Actions: lint + typecheck + test + astro build
  → rsync dist/ → /srv/sites/fw.copyzen.com.br/ (KVM4)
  → nginx:alpine serve os arquivos
  → Traefik: HTTPS + SSL Let's Encrypt
  → fw.copyzen.com.br acessível
```
