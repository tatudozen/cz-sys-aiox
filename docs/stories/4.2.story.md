# Story 4.2: Carrossel Subsystem

## Story

**As a** operator (Fernando),
**I want** the system to generate carousel posts with multiple slides,
**so that** clients get ready-to-publish carousel content for social media.

---

## Status

Approved

---

## Context

**Epic 4 — Sistema 1: Geração de Conteúdo**

Dependência de 3.4 (CopywriterAgent com `generatePostCopy`). Expande a geração de posts para suportar o formato carrossel com múltiplos slides e estrutura específica por modo.

---

## Acceptance Criteria

- [ ] **AC-1:** `CopywriterAgent.generateCarouselCopy(brief, brandConfig, mode, slideCount)` produz:
  - Slide 0 (capa): headline + hook forte
  - Slides 1 a N-1: copy estruturado com conteúdo progressivo
  - Slide final (CTA): Inception = CTA suave | Atração Fatal = CTA forte → link FunWheel

- [ ] **AC-2:** `DesignerAgent.generateCarouselVisuals(slides, brandConfig)` produz prompts de imagem por slide mantendo coerência visual sequencial

- [ ] **AC-3:** Output por slide:
  ```typescript
  interface CarouselSlide {
    index: number;
    copy_text: string;
    image_prompt: string;
    image_url: string | null;
    layout_hint: 'cover' | 'content' | 'cta';
  }
  ```

- [ ] **AC-4:** Modo Inception: slides educativos/branding, sem CTA direto até slide final (CTA suave)

- [ ] **AC-5:** Modo Atração Fatal: tensão crescente por slide, slide final com CTA forte → FunWheel URL

- [ ] **AC-6:** Testes: gerar carrossel 5 slides em cada modo → validar estrutura → brand compliance

## Dev Notes

### Estrutura de Carrossel para Instagram
[Source: PRD#FR-06, 4.2-AC1]
- Carrossel: até 10 slides no Instagram
- Capa (slide 0): texto curto e impactante, visual dominante
- Slides intermediários: copy conciso (max 3 linhas), visual ilustrativo
- Slide final: CTA claro, link na bio (Instagram) ou swipe up (se disponível)

### Prompt Template de Carrossel
```
Crie um carrossel para Instagram de {slideCount} slides no modo {mode}.
Slide 0 (CAPA): headline poderosa que para o scroll
Slides 1-{N-1}: cada slide aprofunda um ponto específico sobre {topic}
Slide final: {inception: "convite suave" | atracao_fatal: "CTA para {funwheel_url}"}
Tom de voz: {brand_guardrails}
```
